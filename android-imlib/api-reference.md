融云即时通讯基础能力库 5.X 版本（IMLib 5.X）分为 6 个子模块：

- `LibCore`：基础核心功能
- `chatroom`：聊天室
- `customservice`：客服
- `discussion`：讨论组
- `locationlib`：实时位置
- `publicservice`：公众号


IMLib 核心类 `RongIMClient` 直接调用各个子模块的接口，内部没有任何具体实现。

以下是 IMLib 5.X 主要模块的 API 参考文档（JavaDoc）：

- [IMLib（通讯库）](https://doc.rongcloud.cn/apidoc/imlib-android/latest/zh_CN/html/)
- [IMLibCore（通讯核心库）](https://doc.rongcloud.cn/apidoc/imlibcore-android/latest/zh_CN/html/)
- [ChatRoom（聊天室库）](https://doc.rongcloud.cn/apidoc/chatroom-android/latest/zh_CN/html/)

下列 IMLib API 不再推荐使用，相关文档仅供参考：

- [CustomService（客服）](https://www.rongcloud.cn/docs/api/android/im_v5/latest/customservice/)
- [PublicService（公众号）](https://www.rongcloud.cn/docs/api/android/im_v5/latest/publicservice/)
- [Discussion（讨论组）](https://www.rongcloud.cn/docs/api/android/im_v5/latest/discussion/)
